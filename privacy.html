隱私權政策

最後更新日期：2026 年 5 月 14 日

本應用程式「個人智慧助理」（以下簡稱「本應用程式」）由 whitedays1051-bot 開發，僅供個人使用。本政策說明本應用程式如何收集、使用及保護您的資料。

1. 收集的資料

本應用程式在您授權後，會透過 Google OAuth 2.0 存取以下資料：

  • Gmail 收件匣郵件內容（僅唯讀，不會修改或刪除）
  • Google 帳號電子郵件地址（用於顯示登入狀態）

2. 資料使用方式

所取得的資料僅用於以下目的：

  • 在應用程式介面中顯示您的 Gmail 信件列表
  • 透過 AI 產生信件摘要，方便您快速閱讀

3. 資料儲存

本應用程式不會將您的 Gmail 內容或 Google 帳號資料儲存於任何伺服器或資料庫。所有資料僅暫存於您的瀏覽器記憶體中，關閉頁面後即自動清除。

4. 資料分享

本應用程式不會將您的任何個人資料分享、出售或提供給第三方。

5. Google API 使用聲明

本應用程式使用 Google API 時，遵守 Google API 服務使用者資料政策，包括有限使用要求。

6. 安全性

本應用程式採用 Google OAuth 2.0 標準授權流程，存取權杖（Access Token）僅存在於瀏覽器記憶體中，不會寫入 Cookie 或本地儲存空間。

7. 使用者權利

您可以隨時前往 Google 帳號安全性設定 撤銷本應用程式的存取權限。

8. 聯絡方式

如有任何疑問，請透過 GitHub 聯繫：github.com/whitedays1051-bot
